## 什么是 Corivo？

Corivo 是一个活在你工作流中的**硅基同事**，不是工具。

- 它悄悄生活在 Claude Code、Cursor、飞书里
- 听着你和 AI 的每一次对话
- 记住那些重要的事
- 在需要的时候温柔地提醒你

**一句话介绍**：你的 Claude 现在有记忆了。

## 安装方式

```bash
curl -fsSL https://corivo.ai | sh
```

安装后说「记住...」就行，它会记住你和 AI 的所有对话。

## 内测说明

### 当前状态

- ✅ **v0.11 开发完成** — 一句话安装、Cold Scan 首次画像、守护进程常驻、自动更新
- 🔄 **内测中** — 正在收集反馈，快速迭代
- ⏳ **公测计划** — 功能稳定后开放公测

### 支持平台

- ✅ macOS arm64（Apple Silicon）- **主要支持平台**
- ⏳ macOS x64（Intel）- 计划中
- ⏳ Linux x64 - 计划中

### 已知限制

1. **守护进程**：目前仅 macOS 支持后台常驻，Linux 用户需手动启动心跳
2. **LLM 集成**：规则引擎可用，但高级功能需要配置 LLM API Key
3. **同步功能**：跨设备同步正在开发中
