# v0.11 Cold Scan 自动执行

> 更新时间：2026-03-20

## 概述

首次启动 `corivo init` 后，现在会**自动执行 Cold Scan**，扫描用户工作环境并构建初始画像。

## 技术实现

### 代码改动

```
packages/cli/src/cli/commands/init.ts
    ├── 导入 coldScan 模块
    ├── 导入 generateFirstPush 模块
    └── 在数据库创建后执行扫描

packages/cli/src/first-push/index.ts
    └── 文案微调："几秒钟" → "一点时间"
```

### 执行流程

```typescript
// 1. 执行 Cold Scan
const scanResult = await coldScan({ verbose: false });

// 2. 保存到数据库（vitality=100, status=active）
for (const block of scanResult.blocks) {
  db.createBlock({
    content: block.content,
    annotation: block.annotation,
    source: block.source,
    vitality: 100,  // 高生命力
    status: 'active',
  });
}

// 3. 生成画像
const storedBlocks = db.queryBlocks({ limit: 1000 });
const firstPush = generateFirstPush(storedBlocks, { minBlocks: 1 });

// 4. 展示给用户
console.log(firstPush.message);
```

## 与实时摄取器的关系

| | Cold Scan | 实时摄取器 |
|---|-----------|------------|
| **触发时机** | 首次 `corivo init` | 持续运行 |
| **数据来源** | 静态文件扫描 | 对话日志、事件流 |
| **更新频率** | 一次性 | 实时 |
| **互补性** | 构建初始画像 | 持续丰富记忆 |

两者**不冲突**，Cold Scan 是"第一次见面握手"，实时摄取器是"日常相处"。
