# Corivo Info

查看当前记忆数据库的运行状态。

## 执行

```bash
corivo status
```

## 输出解读

将状态信息以简洁格式呈现给用户：

- **总数** — 存储的记忆块数量
- **活跃** — 最近被访问/使用的记忆数
- **冷却** — 一段时间未访问的记忆数
- **冷冻** — 长期未使用的记忆数

## 输出格式

```
[corivo] 状态报告：
• 总记忆数：42 块
• 活跃：38 | 冷却：3 | 冷冻：1
• 数据库健康度：90%
```

## 无数据时

```
[corivo] 数据库为空，开始对话即可积累记忆。
```
