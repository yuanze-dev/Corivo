# @corivo/shared

Shared types and API contracts used across Corivo packages and integrations.

## Stability

- Status: `internal beta`
- Scope: support package for monorepo consumers
- Maturity notes: surface area is intentionally small and may evolve with plugin/runtime contracts

## What Is In This Package

- Shared API exports for cross-package use
- Common type definitions published as ESM + type declarations

## Local Development

From this directory:

```bash
npm install
npm run build
npm run dev
```

## Where To Look Next

- Public API entry: `api/index.ts`
- Package metadata and build shape: `package.json`, `tsconfig.json`
